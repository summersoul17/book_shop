# Books API

## Endpoints:

### books
1. `GET /api/books` - список книг
2. `GET /api/books/{book_id}` - выдать информацию по конкретной книге
3. `POST /api/books` - создание книги
4. `DELETE /api/books` - удаление книги
5. `PUT /api/books/{book_id}` - изменение книги
